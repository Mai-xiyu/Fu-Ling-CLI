## 🔧 技术架构

### 项目结构：
```
fuling/
├── fuling_cli.py          # 主入口点
├── ai_cli/               # 原有AI-CLI代码
│   ├── core/            # 核心模块
│   │   ├── ai_providers.py  # 多AI支持
│   │   └── config.py    # 配置管理
│   ├── commands/        # 命令模块
│   └── utils/           # 工具函数
├── config/              # 配置文件
├── plugins/             # 符咒插件
├── tests/               # 测试
└── docs/                # 文档
```

### 命令映射：
```
原AI-CLI → 符灵
ai init      → fl init
ai explain   → fl explain
ai generate  → fl generate
ai chat      → fl chat
ai status    → fl power
ai --help    → fl wisdom
新增: fl fortune (运势)
```
